# Tao UI

Tao UI provide basic global styles and related tools for [Tao](https://github.com/mycolorway/tao_on_rails) project.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'tao_ui'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install tao_ui
```
